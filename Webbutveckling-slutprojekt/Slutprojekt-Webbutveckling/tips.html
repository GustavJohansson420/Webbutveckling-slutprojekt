Tips

När man har lärt sig lite om hur man skriver kod och vilka programmeringsspråk som gör vad så kanske man vill prova att skriva själv. Men hur gör man det då? Här nedanför kan du läsa om appar som kan hjälpa dig med att programmera.

Visual Studio Code

Visual Studio Code är en app som du kan använda dig av för att skriva din kod i. I Visual Studio Code kan man använda sig av många olika genvägar och hjälpmedel som under lättar för dig när du skriver din kod. För att komma till Visual Studio Code's hemsida tryck HÄR. Därifrån kan du sedan ladda ner deras app.

Github

Github är en hemsida där du kan dela din kod och ta inspiration från andra men framför allt så kan du använda dig av Github för att spara din kod där. Om du t.ex jobbar på olika datorer ofta så kan det vara jobbigt att hela tiden behöva föra över det du gjorde på den ena datorn till den andra men detta blir lättare med Github och framför allt med Github's app. För att komm till Githubs hemsida tryck HÄR. Därifrån kan du sedan ladda ner deras app.
