Det går så klart att göra en sida med bara HTML men det är ju ganska tråkigt om all text ser lika dan ut och om ingenting har färg, man vill ju designa sin sida lite. Cascading Style Sheets(CSS) är språket som används för att styla HTML dokument. CSS beskriver hur HTML element ska visas. Kolla järna på tabellen nedan för att lära dig lite om de vanligaste CSS taggarna eller klicka på HTML länken i den lokala menyn till vänster för att lära dig om det och vad man använder det till.

Vanliga CSS properties
Property: Vad den gör: Exempel:
color color: använder man för att ändra färgen på texten p {
color: white;
}
padding padding: använder man för att få utrymme runt någonting p {
padding: 10px;
}
text-align text-align: använder man för att bestämma var man vill ha sin text t.ex i mitten p {
text-align: center;
}
font-size font-size: använder man för att bestämma storleken på sin text. p {
font-size: 30px;
}
