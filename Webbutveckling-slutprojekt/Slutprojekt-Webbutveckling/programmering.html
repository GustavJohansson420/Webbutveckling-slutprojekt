Info:

När man programmerar så instruerar man en maskin eller en del av en maskin som t.ex en dator eller robot att utföra ett visst arbete. När man programmerar skriver man ett program med ett visst programmeringsspråk och dessa finns det många olika av. Klicka på den lokala navigeringsmenyn uppe till höger för att se två populära programmeringspråk som många använder. Maskinen du programmerar förstår oftast inte språket du programmerar på men det översätts till ett maskinspråk som maskinen kommer förstå. Man programmerar nästan aldrig i vanlig maskinkod för det kräver alldeles för mycket arbete.

  2. Python
  3. Java
