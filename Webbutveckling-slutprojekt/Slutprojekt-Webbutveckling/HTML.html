HyperText Markup Language(HTML) är standardspråket för webbsidor. Med hjälp av HTML kan du skapa dina egna hemsidor på internet. HTML är lätt att förstå och lära sig och med hjälp av olika taggar kommer du snabbt lära dig hur man skapar en egen sida på internet. Ta gärna en titt på tabellen nedan för att lära dig lite om de vanligaste taggarna inom HTML och vad de gör eller klick på CSS länken i den lokala menyn till vänster för att lära dig om vad man använder det till.

Vanliga HTML taggar
Tagg: Vad den gör: Exempel:
h1 <h1> gör så man kan skriva huvudtitlen på sin sida och det kan bara finnas en <h1> på varje sida. Sedan finns det <h2>, <h3> osv som också är titlar men dessa är mindre viktiga titlar som kan användas flera gånger. desto högre nummer desto mindre viktig titel. <h1>Här står huvudtitlen på din sida</h1> <h2>Här står titlen på något mindre viktigt</h2>
p <p> gör så man kan skriva vanlig text <p> här står vanlig text som du t.ex kan ha under en h2 tagg</p>
img <img> används för att göra bilder <img src="bilder/hund.png"></img>
a <a> används för att skapa en länk <a href="https://www.facebook.com"></a>
